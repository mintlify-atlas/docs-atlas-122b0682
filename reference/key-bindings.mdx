title: Key bindings and actions

fzf provides a powerful key binding system that allows you to bind keys and events to actions using the `--bind` option.

## Binding syntax

The `--bind` option takes a comma-separated list of binding expressions:

```bash
--bind KEY:ACTION
--bind EVENT:ACTION
--bind KEY1,KEY2,EVENT:ACTION
```

**Examples:**

```bash
fzf --bind=ctrl-j:accept,ctrl-k:kill-line
fzf --bind 'ctrl-a:select-all+accept'
fzf --bind 'ctrl-r:reload(ps -ef)'
```

## Available keys

You can bind actions to these keys:

### Letter and modifier keys

- `ctrl-[a-z]` - Control key combinations
- `ctrl-space` - Control + space
- `ctrl-\` - Control + backslash
- `ctrl-]` - Control + right bracket
- `ctrl-^` or `ctrl-6` - Control + caret
- `ctrl-/` or `ctrl-_` - Control + slash
- `alt-[*]` - Alt + any single character
- `ctrl-alt-[a-z]` - Control + Alt combinations

### Function keys

- `f1` through `f12` - Function keys

### Navigation keys

- `up`, `down`, `left`, `right` - Arrow keys
- `home`, `end` - Home and End keys
- `page-up` or `pgup`, `page-down` or `pgdn` - Page navigation
- `insert` - Insert key
- `delete` or `del` - Delete key

### Special keys

- `enter` or `return` - Enter key
- `space` - Space bar
- `backspace`, `bspace`, or `bs` - Backspace
- `tab` - Tab key
- `shift-tab` or `btab` - Shift + Tab
- `esc` - Escape key

### Shift combinations

- `shift-up`, `shift-down`, `shift-left`, `shift-right`
- `shift-home`, `shift-end`
- `shift-delete`, `shift-page-up`, `shift-page-down`

### Alt combinations

- `alt-enter` - Alt + Enter
- `alt-space` - Alt + Space
- `alt-up`, `alt-down`, `alt-left`, `alt-right`
- `alt-home`, `alt-end`, `alt-delete`
- `alt-backspace`, `alt-bspace`, or `alt-bs`
- `alt-page-up`, `alt-page-down`
- `alt-shift-up`, `alt-shift-down`, etc.

### Ctrl combinations

- `ctrl-up`, `ctrl-down`, `ctrl-left`, `ctrl-right`
- `ctrl-home`, `ctrl-end`, `ctrl-delete`
- `ctrl-backspace`, `ctrl-bspace`, or `ctrl-bs`
- `ctrl-page-up`, `ctrl-page-down`
- `ctrl-shift-up`, `ctrl-shift-down`, etc.

### Multi-modifier combinations

- `ctrl-alt-up`, `ctrl-alt-down`, etc.
- `ctrl-alt-shift-up`, `ctrl-alt-shift-down`, etc.

### Mouse events

- `left-click`, `right-click`, `double-click`
- `scroll-up`, `scroll-down`
- `shift-left-click`, `shift-right-click`
- `shift-scroll-up`, `shift-scroll-down`
- `preview-scroll-up`, `preview-scroll-down`

<Note>
  Some terminal emulators may not support all key combinations, especially `ctrl-*` bindings.
</Note>

## Available events

Events are triggered at specific times during fzf execution.

### start

Triggered once when fzf starts.

```bash
# Move cursor to the last item and select all
seq 1000 | fzf --multi --sync --bind start:last+select-all
```

<Info>
  The input list is not available unless you use `--sync`.
</Info>

### load

Triggered when the input stream is complete and initial processing is done.

```bash
(seq 10; sleep 1; seq 11 20) | fzf --prompt 'Loading> ' \
  --bind 'load:change-prompt:Loaded> '
```

### change

Triggered whenever the query string is changed.

```bash
# Move cursor to the first entry when query changes
fzf --bind change:first
```

### result

Triggered when filtering is complete and the result list is ready.

```bash
# Put cursor on second item when query is empty
fzf --sync --bind 'result:transform:[[ -z {q} ]] && echo "pos(2)"'
```

### focus

Triggered when focus changes due to cursor movement or search result update.

```bash
fzf --bind 'focus:transform-preview-label:echo [ {} ]' --preview 'cat {}'
```

<Warning>
  Actions bound to `focus` run synchronously and can make the interface sluggish. Avoid slow commands.
</Warning>

### resize

Triggered when terminal size changes.

```bash
fzf --bind 'resize:transform-header:echo Resized: ${FZF_COLUMNS}x${FZF_LINES}'
```

### one

Triggered when there's only one match.

```bash
# Automatically select the only match
seq 10 | fzf --bind one:accept
```

### zero

Triggered when there's no match.

```bash
# Reload when there's no match
echo $RANDOM | fzf --bind 'zero:reload(echo $RANDOM)+clear-query'
```

### multi

Triggered when the multi-selection has changed.

### backward-eof

Triggered when trying to delete backward while query is already empty.

```bash
fzf --bind backward-eof:abort
```

### jump

Triggered when successfully jumped to target item in jump mode.

```bash
fzf --bind space:jump,jump:accept
```

### jump-cancel

Triggered when jump mode is cancelled.

```bash
fzf --bind space:jump,jump:accept,jump-cancel:abort
```

### click-header

Triggered when clicking within the header. Sets `FZF_CLICK_HEADER_LINE`, `FZF_CLICK_HEADER_COLUMN`, `FZF_CLICK_HEADER_WORD`, and `FZF_CLICK_HEADER_NTH` variables.

```bash
ps -ef | fzf --header-lines 1 --bind 'click-header:transform-nth(
  echo $FZF_CLICK_HEADER_NTH
)+transform-prompt(
  echo "$FZF_CLICK_HEADER_WORD> "
)'
```

### click-footer

Triggered when clicking within the footer. Sets `FZF_CLICK_FOOTER_LINE`, `FZF_CLICK_FOOTER_COLUMN`, and `FZF_CLICK_FOOTER_WORD` variables.

## Action composition

You can chain multiple actions using the `+` separator:

```bash
fzf --multi --bind 'ctrl-a:select-all+accept'
fzf --bind 'ctrl-r:clear-query+reload(ps -ef)'
```

<Note>
  Any action after a terminal action (like `accept` or `abort`) is ignored.
</Note>

## Action arguments

Actions with `(...)` take arguments. If arguments contain parentheses, use alternative notations:

```bash
action-name(...)  # Standard
action-name[...]  # Alternative
action-name{...}  # Alternative
action-name<...>  # Alternative  
action-name~...~  # Alternative
```

**Example:**

```bash
fzf --bind 'ctrl-o:execute(vim {})'
fzf --bind 'ctrl-o:execute[vim {}]'  # Same thing
```

## Advanced examples

### Multi-action binding

```bash
fzf --multi --bind 'ctrl-a:select-all+accept'
```

### Reload with query

```bash
fzf --bind 'change:reload:sleep 0.1; rg {q} || true' --disabled
```

### Interactive ripgrep

```bash
rg_prefix='rg --column --line-number --no-heading --color=always'
: | fzf --bind 'start:reload:$rg_prefix ""' \
        --bind 'change:reload:$rg_prefix {q} || true' \
        --bind 'enter:become(vim {1} +{2})' \
        --ansi --disabled
```

### Toggle preview

```bash
fzf --preview 'cat {}' \
    --bind 'ctrl-/:change-preview-window(down|hidden|)'
```

### Dynamic labels

```bash
fzf --bind 'focus:transform-preview-label:echo [ {} ]' \
    --preview 'cat {}'
```

### Copy to clipboard

```bash
fzf --bind 'ctrl-y:execute-silent(echo -n {} | pbcopy)+abort'
```
